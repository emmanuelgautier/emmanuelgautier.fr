Parfois, il est nécessaire de changer l'extension d'un ensemble de fichiers d'un dossier. Vous pourrez trouver quelque scripts permettant de le faire pour différentes situations.

## Markdown vers MDX

Si vous voulez changes l'extension des fichiers de `markdown` avec l'extension `.md` vers `.mdx`:

```shell rename-multiple-files-extensions-markdown.sh
for x in **/*.md; do mv "$x" "${x%.md}.mdx"; done
```

## Javascript vers Typescript

Si vous voulez transformer votre projet Javascript avec des fichiers `js` vers un projet Typescript avec des fichiers `.ts`:

```shell rename-multiple-files-extensions-typescript.sh
for x in **/*.js; do mv "$x" "${x%.md}.ts"; done
```
