Tout comme linux, Windows permet de créer entre autre des liens matériels (**hardlink**) et des liens symboliques (**symlink**) en ligne de commande. La ligne de commande Mklink est l’équivalent sous Windows de la célèbre commande `ln` sous linux. Voici, plus bas, une série d’exemple de son utilisation.

Attention cette commande semble ne pas marcher sous le powershell. Il vous faut utiliser la ligne de commande " classique " (Windows + R puis cmd).

Avant tout, si vous ne connaissez pas la différence entre les différents types de liens, je vous invite à consulter les documentations MSDN de Microsoft sur les [liens symboliques](<http://msdn.microsoft.com/en-us/library/aa363878(v=vs.85).aspx>) et sur les [liens materiels et jonctions](http://msdn.microsoft.com/en-us/library/aa365006%28VS.85%29.aspx).

### Lien symbolique

```bash
mklink /D dir_link dir_target
```

### Lien materiel

```bash
mklink /H link dir
```

### Jonction

```bash
mklink /J dir_link dir_target
```
