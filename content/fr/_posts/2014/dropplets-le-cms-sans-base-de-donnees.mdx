Dropplets est un CMS sans base de données ecrit en PHP par [Jason Schuller](http://jason.sc/) . Vous pouvez retrouver les sources de celui-ci sur [github](https://github.com/circa75/dropplets). Vous pouvez l'installer sur n'importe quel serveur supportant le PHP. L'écriture des articles se fait hors-ligne en langage MarkDown. Pour publier un article, il vous suffit de l'uploader avec la / les images qui lui sont rattaché. Comme tout CMS, celui-ci possède des options de personnalisations comme le titre, la phrase d'accroche ou encore l'installation de template. Pour ces derniers, leur nombre est assez faible mais rien ne vous empêche de développer le votre.

[Site Web](http://dropplets.com/)

### Installation

Passons maintenant à l'installation. Pour cela rien de plus simple, récupérez l'archive zip en cliquant sur download sur le site officiel. Extrayez les sources et déposez les sur le serveur. Maintenant, allez à L’URL pour accéder au site. Renseignez le mot de passe administrateur et connectez-vous. Voilà, l'installation est terminé.

### Publication

Pour la publication, l'écriture de l'article se fait dans le langage MarkDown. Ecrivez votre article, puis cliquez sur " Publish or Update Posts " et uploader votre fichier markdown.

```markdown
# Dropplets

- Emmanuel Gautier
- emmanuelgautier
- 2014/03/08
- CMS
- pusblished

Dropplets est une plateforme dédiée à la conception de blog de façon simple. Sans base de données, vous pouvez installer Dropplets en quelques secondes sur n'importe quel serveur, écrire hors-ligne en markdown, puis envoyer vos articles à la publication de manière simple.
```

Et voilà, vous connaissez maintenant un CMS sans base de données !
