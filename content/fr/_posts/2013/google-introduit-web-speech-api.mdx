L'arrivé de la dernière version de Google Chrome signe l'arrivé de la reconnaissance vocale dans les pages web. En effet, le navigateur Google inclut depuis la version 25 le Web Speech API, une interface de programmation permettant la reconnaissance vocale. Elle pourra permettre d'écrire des textes sans avoir à toucher au clavier ! Impressionnant, non ? Une page de test a été mise en ligne par Google : [https://www.google.com/intl/en/chrome/demos/speech.html](https://www.google.com/intl/en/chrome/demos/speech.html).

![](/images/webspeechapi2.jpg)

Mais Google n'avait pas déjà mis en place la reconnaissance vocale ? Si bien sûr, elle était présente sous forme d'une API Javascript notamment sur la page de traduction de Google (Google Translate). Depuis Août 2010, Google, Microsoft et la fondation Mozilla travaillaient sur Web Speech API au sein du groupe HTML Speech Incubator. Après la publication d'un document en 2011 spécifiant les utilisations de la reconnaissances vocale, Google a publié cette API javascript qu'on a pue voir / utiliser sur sa page de traduction.

En avril dernier, un pas a été franchi avec la normalisation de Web Speech API par la W3C. Aujourd'hui, Google l'implémente dans son navigateur et sûrement que demain d'autres navigateurs suivront.

Mais à quoi cela va servir ? A beaucoup de choses ! Vous pourrez, en autre, désormais dicter votre message à votre navigateur, envoyer des mails par la diction. Peut être même naviguer sur le web à la voix. Peut être très pratique notamment pour les aveugles. En voici une petite
video faite par google pour montrer la puissance de cette API :
